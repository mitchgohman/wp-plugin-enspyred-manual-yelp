# WordPress.org Assets

This directory contains assets for the WordPress.org plugin repository.

## Required Assets

### Plugin Icon

- **icon-128x128.png** - Small icon (128x128px)
- **icon-256x256.png** - Large icon (256x256px, retina)

### Plugin Banner

- **banner-772x250.png** - Banner image (772x250px)
- **banner-1544x500.png** - Retina banner (1544x500px)

### Screenshots

Place screenshot files here with sequential naming:

- **screenshot-1.png** - Contact form on frontend
- **screenshot-2.png** - Contact forms management
- **screenshot-3.png** - Plugin settings page
- **screenshot-4.png** - Form configuration options

## Image Guidelines

- Use PNG format for best quality
- Ensure images are optimized for web
- Icons should have transparent backgrounds
- Banners should be visually appealing and represent the plugin

## Deployment

These assets will be automatically copied to the WordPress.org SVN `/assets/` directory during deployment by GitHub Actions.
